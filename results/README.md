# Results

This directory contains generated model outputs from the current ANN workflow.

- `model_metrics.csv`: MAE, RMSE, R2, and MAPE for train, validation, and test splits.
- `prediction_examples.csv`: example measured and predicted compressive strength values.
- `training_history.csv`: epoch-level training and validation metrics.
- `figures/`: generated visualizations used to inspect the dataset and model behavior.
